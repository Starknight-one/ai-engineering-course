# ADW - Automation & Development Workflows

Директория для Python скриптов автоматизации и интеграции с Claude Code.

## Назначение

Здесь хранятся Python скрипты, которые автоматизируют различные задачи через slash команды Claude Code.

## Структура

```
ADW/
├── README.md           # Этот файл
└── (скрипты будут добавлены позже)
```

## Примеры использования

Python скрипты в этой директории могут:
- Запускать последовательности slash команд Claude Code
- Автоматизировать повторяющиеся задачи разработки
- Интегрироваться с внешними API и сервисами
- Генерировать код или конфигурации
- Выполнять batch операции над проектом

## Требования

Для работы скриптов потребуется:
- Python 3.8+
- pip для установки зависимостей

## Установка зависимостей

```bash
cd ADW
pip install -r requirements.txt  # когда файл будет создан
```

## Запуск скриптов

```bash
cd ADW
python script_name.py
```

## Планируемые скрипты

- Автоматизация deploy процессов
- Генерация документации
- Массовые обновления кода
- Интеграционные тесты
- CI/CD автоматизация
